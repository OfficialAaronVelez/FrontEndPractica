Terminal Portfolio

A terminal-themed portfolio website with an command-line interface style.

Features

- Terminal windows with minimize/maximize controls
- Sections: About Me, Technical Skills, Projects, Contact
- Interactive command input with easter egg 
- Responsive design

Getting Started

Open `index.html` in any web browser. No dependencies required.

Technologies

HTML, CSS, JavaScript
